# Lawami Real Estate

A luxury real estate platform for Riyadh, Saudi Arabia, featuring natural language property search, premium listings, mortgage calculators, and an admin CRM panel.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   `npm run dev`
